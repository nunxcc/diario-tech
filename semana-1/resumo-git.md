# Publicar um Repositório no GitHub

1. Criar a pasta do projeto  
2. Entrar nela e inicializar o Git → `git init`  
3. Criar e adicionar ficheiros → `git add .`  
4. Fazer o primeiro commit → `git commit -m "Primeiro commit"`  
5. Criar o repositório no GitHub (sem README)  
6. Ligar o repositório local →  
   `git remote add origin <link-do-repo>`  
7. Definir a branch principal →  
   `git branch -M main`  
8. Fazer o push inicial →  
   `git push -u origin main`
